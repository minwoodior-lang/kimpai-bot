🔹 (1) 해외 미상장 코인 표시 개선 요청

기준 거래소에는 상장되어 있지만, 선택한 해외 거래소에는 미상장인 코인의 경우
현재 하단 값이 ₩0, 0.0000 처럼 표시되고 있습니다.

이 케이스는 “실제 값 0”이 아니라 데이터 없음(해외 미상장) 상태이므로,
아래 조건 중 하나라도 충족하면 숫자 대신 - 로 표기되도록 수정해 주세요.

해외 미상장 판정 조건 (예시):

foreignPriceKrw <= 0

해외 거래소 티커 매칭 실패로 데이터가 들어오지 않는 경우

volume24hForeignKrw === 0 이면서 프리미엄 계산도 불가능한 경우

위 조건에 해당하면, 해당 코인의 “해외 관련 값”은 모두 - 로 통일해 주세요.

현재가 하단: foreignPriceKrw

김프 상단/하단: premiumRate, premiumDiffKrw

전일대비 하단: changeAbsKrw

고가대비·저가대비 하단: highDiffKrw, lowDiffKrw

거래액(일) 하단: volume24hForeignKrw

즉, 해외 미상장 코인은 하단 전체가 - 로 보이게 해 주세요.

🔹 (2) TwoLineCell 스타일/정렬 통일 요청

현재 2줄 구조 자체는 동작하지만,
하단 해외 값의 색상·폰트·정렬이 컬럼마다 조금씩 달라 보입니다.
다음과 같이 TwoLineCell 공통 스타일을 한 번에 교통정리해 주세요.

2-1. TwoLineCell 레이아웃 통일

모든 숫자 셀을 아래 구조로 맞춰 주세요:

<div className="flex flex-col items-end leading-tight">
  <span className="text-sm font-medium">
    {/* 상단 값 (국내 기준 메인 값 / % 등) */}
  </span>
  <span className="text-xs text-gray-500">
    {/* 하단 값 (해외 기준 보조 정보, 또는 '-' ) */}
  </span>
</div>


items-end + text-right 로 상·하단 모두 오른쪽 정렬 고정

leading-tight 또는 고정된 gap 으로 줄간 간격 일정하게 유지

상단은 기존 스타일 그대로(국내 기준 값 / % 값에 녹색·빨강 적용)

하단은 항상 회색 텍스트(해외 현재가 스타일) 로 통일

2-2. 색상 규칙

상단(국내 기준 메인 값)

현재가: 기존처럼 흰색/메인 색상 유지

김프·전일대비·고가대비·저가대비:

양수 → 녹색

음수 → 빨간색

0 또는 데이터 없음 → 중립색

하단(해외 기준 보조 값 전체)

항상 회색(예: text-gray-500 + text-xs) 으로 표시

상승/하락 여부에 따라 색 바꾸지 않음

미상장(-)도 동일한 회색 스타일로 보여줌

→ 이렇게 하면 상단은 “국내 메인 정보”, 하단은 “해외/보조 정보”가 색상만 봐도 명확히 구분됩니다.

2-3. 기존 2줄 매핑은 유지

현재 적용된 컬럼 매핑은 그대로 유지해 주세요:

현재가: 상단 = koreanPrice, 하단 = foreignPriceKrw

김프: 상단 = premiumRate, 하단 = premiumDiffKrw

전일대비: 상단 = changeRate, 하단 = changeAbsKrw

고가대비: 상단 = fromHighRate, 하단 = highDiffKrw

저가대비: 상단 = fromLowRate, 하단 = lowDiffKrw

거래액(일): 상단 = volume24hKrw, 하단 = volume24hForeignKrw

이 두 가지(1. 해외 미상장 - 처리, 2. TwoLineCell 스타일·색상·정렬 통일)까지 반영되면,
지금 홈 테이블 UX는 김프가 스타일로 거의 완성이라고 보면 됩니다.

🔸 추가 조건 – 깜빡임 유지
하단 해외 값은 기본 텍스트 색상만 회색으로 통일하고,
가격이 변동될 때 적용되는 기존 깜빡임 효과(상승/하락 시 초록/빨강으로 잠깐 플래시 후 다시 원래 색으로 복귀)는 상단·하단 모두 그대로 유지해 주세요.
즉, 기본 색 → 회색, 플래시 애니메이션 로직은 기존 PriceCell 동작을 그대로 사용하면 됩니다.
