1. PC 카드 – PRO 카드랑 1:1 복붙 수준으로 맞추기
1) 버튼 색/두께 완전 동일하게

지금 오른쪽 버튼이 PRO 버튼보다 연해 보이고, 폰트가 살짝 얇아 보여.

👉 요청 포인트 한 줄로:
ProPredictionCard 버튼 JSX 통째로 복사해서,
`href/텍스트/아이콘만 바꾸고 className은 한 글자도 바꾸지 말고 그대로 쓰게.

전달문용 예시:

// ProPredictionCard 에서 쓰는 버튼 JSX 그대로 가져오기
<Link
  href={TELEGRAM_FREE_SIGNAL_URL} // 링크만 변경
  target="_blank"
  rel="noopener noreferrer"
  className="mt-auto inline-flex items-center justify-center gap-2 rounded-xl bg-[#7C3AED] hover:bg-[#8B5CF6] text-sm font-semibold text-white py-2" 
  // ↑ 이 줄은 ProPredictionCard 버튼 className이랑 100% 동일하게 맞출 것
>
  <svg ... className="w-4 h-4" />
  텔레그램 채널 열기
</Link>


className 에서 bg-..., hover:bg-..., text-sm, py-2 이런 거
한 글자라도 PRO랑 다르면 안 된다고 못 박아 두면 됨.

2) 타이틀/본문 폰트 크기

지금 오른쪽 카드는:

타이틀: PRO보다 1단계 작아 보이고

본문: 너무 작게 들어가 있어서 카드가 텅 빈 느낌.

👉 요청:

타이틀 <div> 의 className 을 그대로 ProPredictionCard 타이틀에서 복붙

본문 <p> 도 ProPredictionCard에서 “작은 설명문”에 쓰는 className 그대로 사용

예시 전달:

// 타이틀 – ProPredictionCard 타이틀 className 그대로
<div className="text-base font-semibold text-slate-100 mb-2">
  📡 실시간 시그널 채널
</div>

// 본문 – ProPredictionCard 본문 className 그대로
<p className="text-sm text-slate-300 leading-relaxed">
  Binance 고래 매매 · 거래량 폭발 · BTC/ETH 김프 급변만 실시간 발송.
</p>

2. 모바일 카드 – PRO 모바일 카드랑 완전 쌍둥이로

모바일도 지금 구조는 괜찮은데,

타이틀/본문 폰트가 PRO보다 살짝 작고

버튼 className 이 PRO 모바일 버튼이랑 미묘하게 다를 확률 큼.

👉 요청 요약:
ProForecastMobileContent 에서 타이틀/본문/버튼 className 을 그대로 가져와서
MyAlertsMobileContent 에 적용하라고 해.

전달용 예시:

export function MyAlertsMobileContent() {
  return (
    <div className="rounded-2xl bg-slate-900/80 border border-slate-800 p-4 space-y-3">
      {/* 타이틀 – ProForecastMobileContent 타이틀 className 복붙 */}
      <div className="text-sm font-semibold text-slate-100">
        📡 실시간 시그널 채널
      </div>

      {/* 본문 – ProForecastMobileContent 본문 className 복붙 */}
      <p className="text-xs text-slate-300 leading-relaxed">
        Binance 고래 매매 · 거래량 폭발 · BTC/ETH 김프 급변만 실시간 발송.
      </p>

      {/* 버튼 – ProForecastMobileContent 버튼 className 복붙 */}
      <Link
        href="https://t.me/kimp_ai"
        target="_blank"
        rel="noopener noreferrer"
        className="inline-flex w-full items-center justify-center gap-2 rounded-xl bg-[#7C3AED] hover:bg-[#8B5CF6] text-xs font-semibold text-white py-2"
      >
        <svg ... className="w-4 h-4" />
        텔레그램 채널 열기
      </Link>
    </div>
  );
}


핵심은 “Pro 버튼/텍스트 className 그대로 → copy/paste”야.
새로운 tailwind 클래스 쓰지 말라고 한번 더 못 박아 두면 알아들을 듯.
