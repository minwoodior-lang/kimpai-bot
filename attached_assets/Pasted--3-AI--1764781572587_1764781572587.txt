┌───────────────────────────────────────────────────────────────────┐

지금 개발자가 “센터 PRO 카드”를 아래로 빼버렸는데, 그게 틀린 것이고, 형이 원한 건 아래 이미지와 동일한 우측 카드 스타일의 중앙 배치임.

형이 원하는 건 딱 이 모습임:

📌 센터에 들어갈 PRO 전용 48시간 김프 예측 카드(미러 처리 포함)

형이 마지막으로 올린 AI 분석 페이지 UI(이거👇)


이 카드를 그대로 복제해서 중앙에 넣어야 해.
회색 블러 뒷배경 + 자물쇠 아이콘 + PRO 버튼 포함해서 그대로.

📌 우측에 들어갈 "내 알림" 카드 구조

지금 우측 “오전 02:01 기준 | 내 알림” 자리에
아래와 같은 카드가 와야 함:

────────────────────────
🔔 내 알림

- 김프 3% 이상 상승 시 알림 설정됨(+)
- 김프 급락 구간 경보 예측 사용 가능
(로그인 안 했을 시 "로그인하고 알림 설정하기")
────────────────────────

📌 이제 개발자에게 넘길 최종 지시문 (복붙하면 됨)

이건 개발자가 절대 틀릴 수 없는 수준으로 내가 정교하게 작성한 지시문이다.

📄 [홈 화면 상단 3단 UI 재구성 작업 지시 – 최종본]
홈 상단 레이아웃은 아래 3개 컬럼으로 구성해주세요:

-------------------------------------------------------------------
| 1) 좌측 컬럼: 오늘의 AI 김프 요약                               |
|   - 현재 구현된 컴포넌트 그대로 유지                             |
|   - 폭: 33%                                                      |
-------------------------------------------------------------------
| 2) 중앙 컬럼: PRO 전용 48시간 김프 예측 카드                    |
|   - ‘AI 분석 페이지 우측의 PRO 카드’ UI 를 그대로 복제합니다.    |
|   - 구성:                                                        |
|       🔒 PRO 전용 48시간 김프 예측 (타이틀)                      |
|       흐릿한 블러 처리 배경 + 예측 설명 텍스트 2~3줄             |
|       아래에는 “PRO 분석 전체 보기” 버튼                         |
|   - 폭: 33%                                                      |
-------------------------------------------------------------------
| 3) 우측 컬럼: 내 알림 카드                                      |
|   - 상단에 이미 존재하는 '오전 02:01 기준 | 내 알림' 텍스트 유지 |
|   - 아래에 아래 형태의 카드 삽입:                               |
|       🔔 내 알림                                                 |
|       - 김프 경보 알림 목록 또는 설정 버튼                       |
|       (로그인 안 했을 시 “로그인하고 알림 설정하기” 문구)        |
|   - 폭: 33%                                                      |
-------------------------------------------------------------------

📌 전체는 grid 또는 flex 로 3등분 배치하며,
각 컬럼은 동일한 높이를 유지해야 합니다.

📌 지금 화면에 있는 오래된 텍스트 박스(“최근 30일 기준 … 90%”)는
반드시 삭제하고, 중앙 PRO 카드 안에서만 사용해야 합니다.

📌 PRO 카드의 스타일(배경, 블러, 그림자, 라운딩, 버튼 색상)은
AI 분석 페이지 우측의 PRO 카드 스타일과 100% 동일하게 구현해야 합니다.
