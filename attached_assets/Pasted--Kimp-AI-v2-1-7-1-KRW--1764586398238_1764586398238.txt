[요청 1] 거래액(일) 셀 표기 방식 개선 (국내/해외 둘 다 KRW 메인)

현재:

거래액(일) 셀에 국내/해외가 섞여 보이고, 아래쪽 USDT 표기가 눈에 많이 띄는 편입니다.

목표:

국내·해외 모두 “KRW 기준”으로 한눈에 비교 가능하게 하고

USDT 값은 제일 아래 작게 보조 정보로만 표기합니다.

구현 가이드 (PremiumTable.tsx):

// 이미 계산된 값이 있다고 가정
const domesticVolumeKrw = row.domesticVolumeKrw;     // Upbit acc_trade_price_24h
const overseasVolumeKrw = row.overseasVolumeKrw;     // quoteVolume * usdtKrw
const overseasVolumeUsdt = row.overseasVolumeUsdt;   // 24h quote volume (USDT)

// 기존 거래액 셀 대체
<td className="px-4 py-3 text-right align-middle">
  <div className="flex flex-col items-end leading-tight">
    {/* 국내 거래액 */}
    <span>
      {formatKrwVolume(domesticVolumeKrw)}
      <span className="ml-1 text-xs text-muted-foreground">
        (국내)
      </span>
    </span>

    {/* 해외 거래액 – KRW 기준 */}
    <span>
      {formatKrwVolume(overseasVolumeKrw)}
      <span className="ml-1 text-xs text-muted-foreground">
        (해외)
      </span>
    </span>

    {/* USDT 보조 표기 – 아주 작게 */}
    <span className="mt-0.5 text-[11px] text-muted-foreground">
      {formatUsdtVolume(overseasVolumeUsdt)} USDT
    </span>
  </div>
</td>


formatKrwVolume : 기존에 쓰는 ₩5,114.0억 / ₩3.7조 포맷 그대로 사용

formatUsdtVolume : $66.89B / $3.77B 등 2자리 소수 + B/M/K

검증 기준:

BTC 기준으로

1행: 업비트 거래대금(24h) KRW ≈ 업비트 공식 화면 값

2행: 바이낸스 거래대금(24h) KRW

두 줄 모두 KRW 기준이라 직관적으로 비교 가능해야 합니다.

[요청 2] 상단 우측 환율 라벨 수정 (USD → USDT 표기 명확화)

현재:

우측 상단에 USD/KRW ₩1,467.9 로 표기되어 있습니다.

실제 의미:

내부적으로 테더(USDT) 환율을 기준으로 사용하고 있으므로
USD/KRW 문구는 다소 모호합니다.

수정 사항:

라벨 텍스트 변경

USD/KRW → USDT/KRW

또는 테더 환율 로 변경 (예: USDT/KRW ₩1,467.9)

선택 (가능하면):

hover 시 작은 툴팁: "바이낸스 USDT 호가 기준 환율" 정도 문구 노출

위치/스타일은 기존과 동일하게 유지하고,
표기만 명확히 “USDT 기준 환율”로 보이게 해주세요.
