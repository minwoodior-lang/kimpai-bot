🎯 1. 목표

KimpAI의 실시간 알트/김프 시그널을 **“리포트 스타일의 텔레그램 알림”**으로 제공하는 텔레봇을 개발한다.
단순 수치 알림이 아니라:

AI 분석

과거 패턴 확률

전략 가이드

가 포함된 고급 분석 리포트를 자동으로 보내는 구조로 구현한다.

💡 무료(FREE) 정책

실시간 시장 감지

시총 TOP 50 알트 자동 탐지

AI 리포트 공식 텔레그램 채널에 자동 방송

⭐ PRO 정책

사용자 관심종목 = 모든 코인 가능 (잡알트 포함)

관심종목마다 AI가 자동 감지 후 개인 DM 발송

BTC 48시간 예측 / 고래 매집 / 과열 리스크 모두 PRO 제공

웹사이트는 https://kimpai.io
 기준으로 안내

🧱 2. 전체 봇 구조
무료(FREE) 명령어

/btc → BTC 김프 변화 감지 리포트

/eth → ETH 변동성 증가 신호

/alt {symbol} → TOP50 알트 1종 단기 분석

PRO 명령어

/pro_btc → BTC 48시간 예측 리포트

/pro_whale {symbol} → 고래 매집 포착 리포트

/pro_risk {symbol} → 과열·폭락 리스크 리포트

자동 전송 구조
구분	자동분석 대상	전송 위치
FREE	시총 TOP 50 알트	KimpAI 공식 텔레그램 채널
PRO	사용자가 등록한 관심종목(모든 코인)	사용자 개인 DM(chat_id)

⚙️ 5. 구현 기술 명세
A) Bot Framework

Node.js + Telegraf.js (권장)
or

Python + python-telegram-bot

B) 구조

bot/index.js or bot/main.py

Webhook or Long polling 방식 가능

.env → BOT_TOKEN, CHANNEL_ID, API_URL 등록

C) 스케줄러

FREE: 10분마다 TOP50 자동 스캔 → CHANNEL_ID 로 메시지 전송

PRO: 관심종목 5~10분마다 스캔 → 해당 유저 DM(chat_id)으로 전송
