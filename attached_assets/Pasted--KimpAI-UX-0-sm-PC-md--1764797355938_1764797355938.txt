📌 KimpAI 홈 모바일 UX 수정 통합 지시서
0. 공통 전제

이번 수정은 모바일 전용(sm 이하) 에만 적용합니다.

PC 레이아웃(md 이상)은 기존 구조/크기 유지하고, 텍스트만 필요한 부분만 통일해 주세요.

1. 상단 티커바(TopInfoBar) – 모바일 BTC 문구 수정

현재 모바일 상단 티커바에서 BTC 관련 부분이 BTC 42.3%처럼 표시되고 있습니다.
모바일에서만 아래처럼 텍스트에 점유율을 추가해 주세요.

기존: BTC 42.3%

변경: BTC 점유율 42.3%

구현 방향 예시:

// 모바일 전용 레이아웃 부분에서
<span>{`BTC 점유율 ${btcDominance}%`}</span>


※ PC 버전 텍스트는 그대로 두고, 모바일(sm 이하)에서만 BTC 점유율이 보이도록 처리해 주세요.

2. 홈 상단 탭 카드 (AI 요약 / PRO 예측 / 내 알림) – 높이 & 여백 조정

모바일에서 상단 3개 카드 탭의 내용 대비 여백이 과해서,
카드 전체 높이와 내부 패딩을 줄여서 더 컴팩트하게 보여주고 싶습니다.

2-1. 공통 카드 래퍼 높이 고정 (모바일만)

세 탭(AI 요약 / PRO 예측 / 내 알림)은
이미 “공통 카드 껍데기 하나 + 내부 내용만 변경” 구조로 되어 있으니,
이 공통 래퍼에 모바일 전용 높이를 지정해 주세요.

예시:

// 모바일 카드 공통 래퍼
<div className="rounded-2xl bg-slate-900 p-4 h-[180px] md:h-auto flex flex-col justify-between">
  {active === "summary" && <AiSummaryMobileContent />}
  {active === "pro" && <ProForecastMobileContent />}
  {active === "alerts" && <MyAlertsMobileContent />}
</div>


sm 이하: h-[180px] 정도로 고정 (필요하면 180~190px 범위에서 맞춰주세요)

md 이상: 기존처럼 auto / PC 레이아웃 유지

핵심은:

모바일에서 3개 탭이 모두 완전히 동일한 카드 높이를 가지도록 하는 것

탭 전환 시 아래 차트/테이블이 흔들리지 않도록 하는 것

2-2. 카드 내부 여백 축소

AI 요약 카드 기준으로, 내용 대비 위/아래 여백이 많으니
모바일에서만 내부 패딩/마진을 한 단계 줄여 주세요.

예시:

카드 내부 상하 패딩

py-4 → py-3 정도로 감소

중간 여백 (Score 바 위 margin-top 등)

mt-4 → mt-2 정도로 감소

구체적인 클래스명은 현재 구조를 보면서 조정해 주시되,
“카드가 세로로 약간 더 컴팩트해 보이도록” 해주시면 됩니다.

3. 프리미엄 차트 박스 높이 축소 (모바일만)

홈 화면에서 상단 카드 바로 아래에 있는 프리미엄 차트 영역 역시
모바일에서 상대적으로 너무 큰 빈 박스로 보이기 때문에,
모바일 전용 높이를 줄여 주세요.

예시:

<div className="rounded-2xl bg-slate-900 ... h-[240px] md:h-[320px]">
  {/* 차트 컴포넌트 */}
</div>


sm 이하: h-[240px] 정도

md 이상: 기존 높이(예: h-[320px]) 유지

정확한 값은 현재 UI를 보면서 살짝 조정하셔도 괜찮습니다.
목표는 모바일에서 카드 + 차트가 너무 길게 늘어지지 않게 만드는 것입니다.

4. Score 라벨 텍스트 통일

AI 요약 카드 하단의 Score 바 텍스트를
PC/모바일 공통으로 “KR Premium Score” 기준으로 통일해 주세요.

현재 모바일 카드에는 바 안쪽에 Score만 표시되고 있는데,
의미가 모호하므로 다음처럼 변경하고 싶습니다.

제안안 (예시 1 – 라벨 + 숫자 분리):

<div className="text-[11px] text-slate-300 mb-1">
  KR Premium Score
</div>
<div className="rounded-xl bg-indigo-700 py-2 text-center text-[14px] font-semibold">
  2/10
</div>


또는 (예시 2 – 한 줄 표기):

<div className="rounded-xl bg-indigo-700 py-2 text-center text-[14px] font-semibold">
  KR Premium Score 2/10
</div>


둘 중 구현하시기 쉬운 방식으로 적용해 주시되,

어디에서도 Score라는 단독 텍스트는 쓰지 않고

항상 KR Premium Score라는 이름으로 노출되도록

PC/모바일 공통으로 텍스트를 맞춰 주세요.

이 네 가지가 이번 수정 범위 전부입니다.

모바일 상단 티커에서 BTC → BTC 점유율 텍스트 수정

모바일 상단 3개 카드 공통 높이 h-[180px] 수준으로 축소 + 내부 여백 줄이기

모바일 차트 높이 h-[240px] 수준으로 축소

Score 라벨을 “KR Premium Score” 로 PC/모바일 공통 통일

이대로 적용해주시면, 모바일 홈 UX가 훨씬 타이트하고 의미도 명확해질 것 같습니다.
