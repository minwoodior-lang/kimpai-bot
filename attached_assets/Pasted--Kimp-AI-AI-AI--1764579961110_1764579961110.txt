📌 + 무료/유료 차이점은 AI 기능에서만 발생 (데이터 속도는 동일)

담당자분께,
아래 내용은 Kimp AI의 홈(/) 화면을 김프가(kimpga.com)와 거의 동일한 UX로 복제하면서,
AI 기능만 강화하는 방향의 최종 개발 지시문입니다.

핵심 목표는 다음과 같습니다:

“김프가 쓰던 유저가 우리 사이트에 들어와도 바로 적응할 수 있을 정도로
구조·UI·동작은 완전히 동일.
차이는 AI 분석 기능에서만 발생하도록 한다.”

📌 4. 테이블 상단 컨트롤 바 (김프가 1:1 복제)
국내/해외 드롭다운 독립 선택

국내: 업비트 / 빗썸 / 코인원 / 코빗
해외: 바이낸스 / Bybit / OKX / Bitget / Gate / MEXC / HTX 등

검색 (김프가와 동일)

한글명

초성

심볼

엔터 없이 실시간 필터링

📌 5. 테이블 본문 (김프가와 동일한 레이아웃+동작)
5-1. 컬럼 구성 그대로

코인명 (한글 + 영문심볼)

현재가(국내)

해외가

김프 %

전일대비 (위 % / 아래 원)

고가대비 (위 % / 아래 원)

저가대비 (위 % / 아래 원)

거래액(위 국내 / 아래 해외)

5-2. 정렬 규칙 동일

행 클릭 없음

코인명 클릭 -> 코인마켓캡 새창

열 헤더 클릭으로 오름/내림정렬

5-3. 페이지네이션 없음

전체 리스트 한 번에 렌더링

김프가와 동일하게 스크롤만 길어짐

📌 6. 데이터 갱신(속도 전략: 김프가와 동일)

업데이트 속도: 약 1초~2초 (무료/유료 동일)

Websocket 또는 1~2초 polling

전체 리렌더 ❌

값만 자연스럽게 업데이트

⚠️ 중요한 요구사항

무료/유료 차별은 데이터속도에 두지 않는다.
실시간 시세는 모든 유저에게 동일하게 제공한다.
유료차별화는 “AI 기능”에서만 발생해야 한다.
