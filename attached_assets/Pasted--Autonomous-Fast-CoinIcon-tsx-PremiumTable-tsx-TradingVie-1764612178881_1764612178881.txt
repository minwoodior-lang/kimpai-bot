⚠️ 중요: Autonomous / 자율 모드는 절대 켜지 마.
Fast 모드에서 CoinIcon.tsx 관련 작업만 해줘.
PremiumTable.tsx, TradingView, 다른 파일들은 이번 작업에서 건드리지 마.

프로젝트: KimpAI (김프 AI)
작업 목표: 
- 현재 앱에서 [CoinIcon] missing icon 경고가 찍히는 모든 심볼에 대해,
  가능한 한 "실제 코인 아이콘"이 표시되도록 COIN_ID_MAP과 관련 로직을 정리하는 것.
- 최종적으로는 콘솔에 [CoinIcon] missing icon 로그가 0개가 되는 상태를 만드는 것이 목표다.

현재 상태:
- CoinIcon.tsx에 COIN_ID_MAP과 여러 CDN(cryptocurrency-icons, spothq, coincap, coingecko, cryptocompare)이 있고,
  찾지 못하면 심볼별 그라데이션 폴백으로 떨어지는 구조다.
- Fast 모드로 20개 정도 신규 코인 매핑을 추가했지만,
  여전히 콘솔에는 [CoinIcon] missing icon 경고가 찍히고 있고,
  화면에는 그라데이션만 나오는 코인들이 남아 있다.

이번에 해줬으면 하는 작업:

1) 실제로 어떤 심볼들이 아직 아이콘이 없는지 "정확한 리스트"를 만들어줘
-------------------------------------------------------------
- 개발 모드에서 KimpAI 앱을 실행하고,
  Markets / Premium / 모바일 리스트 등 코인 목록이 노출되는 페이지를 한 번씩 끝까지 스크롤해.
- 이 때 콘솔에 찍히는 `[CoinIcon] missing icon for symbol: XXX` 경고를 전부 수집해서,
  Set<string> 기준으로 중복 제거된 심볼 목록을 만들어줘.
- 이 심볼 목록을 작업 결과에 "명시적으로" 출력해줘. 
  (예: `현재 아이콘이 없는 심볼: [ "0G", "WLFI", "KITE", ... ]` 이런 식)

2) COIN_ID_MAP을 가능한 한 "진짜 아이콘" 기준으로 채워줘
-------------------------------------------------------------
- 1번에서 수집한 모든 심볼에 대해, 다음 순서로 아이콘을 찾는 작업을 해줘:

  1. cryptocurrency-icons / spothq / coincap에서 해당 심볼의 아이콘이 있는지 확인
  2. 없으면 CoinGecko에서 적절한 코인 ID를 찾은 뒤, 그 ID를 COIN_ID_MAP에 매핑
  3. 그래도 아이콘이 전혀 없으면, 마지막 수단으로만 그라데이션 폴백 유지

- 중요한 점:
  - "아이콘이 아예 없는 코인"이 아니라면, 
    가능한 한 반드시 이미지 아이콘을 보여줄 수 있도록 COIN_ID_MAP을 채워줘.
  - 심볼이 겹치거나 여러 프로젝트가 있는 경우(예: A, W, LA, FF 같은 것들)는
    업비트/OKX에 실제 상장된 프로젝트 기준으로 최대한 맞춰줘.
  - 지금 이미 그라데이션이 정의된 심볼이라도,
    CDN이나 CoinGecko에서 아이콘을 찾을 수 있다면
    그라데이션이 아니라 실제 아이콘이 우선 적용되도록 수정해줘.

3) loggedMissingIcons / 디버그 출력 유지 + 최종 검증
-------------------------------------------------------------
- 현재 CoinIcon.tsx에 loggedMissingIcons Set과 dev 전용 디버그 출력이 있는 상태인데,
  이 기능은 그대로 유지해줘.
- 대신, 이번 작업 이후 앱을 다시 실행하고,
  콘솔에 `[CoinIcon] missing icon for symbol:` 경고가 아직 남아 있다면:

  - 그 심볼 목록을 다시 한 번 정리해서
    "정말로 아이콘을 찾을 수 없었던 코인 목록"으로 결과 메시지에 포함해줘.
  - 이 목록은 나중에 내가 별도로 검토하고, 필요하면 수동으로 매핑할 수 있도록 남겨줘.

4) PremiumTable / 429 관련 코드는 건드리지 말 것
-------------------------------------------------------------
- 이미 PremiumTable.tsx에는 3초 폴링 + 429 백오프(10초 딜레이, 5회 이상 429 시 60초 대기)가 도입된 상태다.
- 이번 작업에서는 PremiumTable.tsx를 수정하지 말고,
  오직 CoinIcon.tsx와 COIN_ID_MAP, 아이콘 관련 로직에만 집중해줘.

완료 기준:
- 개발 모드에서 Markets / Premium 페이지를 새로고침 후 끝까지 스크롤했을 때,
  `[CoinIcon] missing icon for symbol:` 로그가 0개이거나,
  정말로 아이콘 소스가 존재하지 않는 극소수 심볼만 남아 있어야 한다.
- 작업 결과로 "아이콘을 끝내 찾지 못한 심볼 리스트"와
  "새로 추가/수정한 COIN_ID_MAP 엔트리들"을 정리해서 알려주면 좋겠다.
