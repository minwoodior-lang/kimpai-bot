3) 코인별 심볼도 실제 심볼로 적용

코인명 옆 아이콘을 실제 원형 심볼로 바꿔주세요.

규칙

48px 또는 32px 원형 아이콘

CoinGecko, Binance API에서 제공하는 심볼 사용

이름 왼쪽에 배치 (김프가 동일 UI)

4) 해외 미상장 코인 처리 규칙 변경

현재 회색 처리 & 차트 없음 → UX 잘못된 상태.

변경 규칙 (김프가와 동일)

행은 정상 스타일 유지 (회색 NO)

해외 가격 / 김프 / 해외 거래대금만 "-"

차트보기 버튼은 존재해야 함
→ 국내 가격 기준으로라도 차트 열려야 함.

코인 리스트는 그대로 유지하고 제거 금지

5) 테이블 기본 정렬 기준 변경 – 거래대금 많은 순(DESC)

현재는 거래액 작은 순으로 정렬됨 → 비정상.

요청

기본 정렬 값 = 거래대금(일) 많은 순으로 정렬

국내+해외 합산 거래액으로 정렬 (가능하면)

불가 시 국내 거래액 기준 DESC

6) Binance API 장애 해결 후 해외 기본 거래소 다시 Binance로 변경 (중요)

현재 region block 때문에 전부 OKX 기반으로 변경됨.

요청

Cloudflare Worker / API Relay 등 사용하여
Binance /api/v3/ticker/24hr 정상 호출되도록 구현

해결 즉시
→ 해외 기본 거래소: Binance USDT 마켓으로 되돌릴 것

Binance 볼륨/가격이 정확해야 플랫폼 신뢰도 확보됨.

7) 가격 소수점 표기 규칙을 김프가와 동일하게 적용

현재 73원 → 소수점 없음 등 표기가 매끄럽지 않음.

가격 표기 규칙
가격대	표기 방식

1원 미만	소수점 4자리
0.0001 미만	지수표기 금지, 0.0000n 형태
