🔹 (1) 해외 현재가 깜빡임 효과 복구 요청

최근 TwoLineCell 적용 이후,
상단 국내 값은 가격 변동 시 깜빡임(flash) 효과가 잘 작동하지만,
하단 해외 현재가는 깜빡임이 작동하지 않는 상태입니다.

의도는 다음과 같습니다:

가격이 변동되면 상·하단 둘 중 어느 값이든 변경될 때

PriceCell 전체에 기존처럼 flash-up / flash-down 애니메이션이 적용되어야 합니다.

기본 색상은 지금처럼 유지:

상단: 기존 색상 규칙 (흰색 + 양수 초록 / 음수 빨강 등)

하단: 기본은 text-gray-500 회색

다만 깜빡이는 순간만

상·하단 모두 초록/빨강으로 잠깐 플래시

이후 다시 상단은 원래 색, 하단은 회색으로 복귀

구현 방향 예시:

PriceCell/TwowLineCell 구조에서

이전 값과 현재 값을 비교하는 로직을
topValue + bottomValue 둘 다 기준으로 보고,

둘 중 하나라도 변경되면 wrapper에 flash-up or flash-down 클래스를 적용

CSS 애니메이션(price-flash-up, price-flash-down)은 기존 것을 그대로 사용하되,
기본 텍스트 색상만 회색으로 바뀐 것일 뿐, 로직은 건드리지 말아 주세요.

요약:

“하단 해외 값도 상단과 동일하게 가격 변동 시 깜빡임은 유지하되,
기본 상태의 텍스트 컬러만 회색으로 보이도록 수정해 주세요.”

🔹 (2) 코인 셀 + 모든 숫자 폰트 크기/스타일 통일 요청

현재 테이블을 보면:

코인명 셀(한글 이름 + 심볼) 폰트 크기/굵기가 행마다 약간씩 다르고,

숫자 컬럼(현재가/김프/전일대비/고가대비/저가대비/거래액)도
상단/하단 글자 크기와 두께가 컬럼마다 미묘하게 달라서
전체적으로 라인이 울퉁불퉁해 보이는 상태입니다.

김프가 UI처럼 깔끔하게 맞추기 위해,
아래와 같이 폰트 크기·굵기를 전역으로 통일해 주세요.

2-1. 코인명 셀(코인 이름 + 심볼)

레이아웃 예시:

<div className="flex flex-col">
  <span className="text-sm font-medium">
    {/* 한글 코인명 (예: 비트코인, 이더리움) */}
  </span>
  <span className="text-xs text-gray-500">
    {/* 심볼 or 영문명 (예: BTC, ETH) */}
  </span>
</div>


모든 코인행에서 동일한 클래스를 사용해서
코인명이 위아래 줄 모두 같은 기준으로 보이게 해 주세요.

2-2. 모든 숫자 셀(TwoLineCell) 공통 스타일

현재 TwoLineCell은 레이아웃은 통일되어 있지만
텍스트 크기/굵기가 컬럼마다 조금씩 다른 부분이 있습니다.
다음처럼 완전히 고정된 공통 스타일로 맞춰 주세요:

<div className="flex flex-col items-end leading-tight">
  <span className="text-sm font-medium">
    {/* 상단 값: 국내 기준 메인 값 / % 값 */}
  </span>
  <span className="text-xs text-gray-500">
    {/* 하단 값: 해외 기준 보조 값 또는 '-' */}
  </span>
</div>


숫자 컬럼 상단: 전부 text-sm font-medium
(현재가, 김프 %, 전일대비 %, 고가대비 %, 저가대비 %, 거래액 상단 모두 동일)

숫자 컬럼 하단: 전부 text-xs text-gray-500
(해외 가격/차액/거래액 등 모두 동일)

items-end + text-right 로 상·하단 모두 오른쪽 정렬 유지

leading-tight 혹은 일정한 gap 으로 줄 간격 고정

요약:

코인명 셀: 위 text-sm font-medium, 아래 text-xs text-gray-500

모든 숫자 셀: 상단 text-sm font-medium, 하단 text-xs text-gray-500

컬럼별로 다른 폰트/크기 쓰지 말고 TwoLineCell/코인셀 공통 스타일로 통일
