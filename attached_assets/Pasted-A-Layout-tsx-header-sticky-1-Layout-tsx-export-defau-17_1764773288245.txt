D. TopMarketInfoBar sticky 동작 (모바일 비고정, PC 고정)

현재 header 전체를 sticky로 만들었으니,
TopMarketInfoBar 자체에는 sticky/fixed 클래스가 있으면 제거해야 한다.

TopMarketInfoBar의 최상단 div 클래스:

<div className="border-b border-slate-900">
  ...
</div>


sticky 관련 클래스 (sticky, top-0, sm:sticky 같은 것) 있으면 전부 삭제.
이제 sticky는 Layout.tsx의 <header>에서만 관리.

E. Chart/HomeLayout 배경 중첩 제거
1) 목표

전체 배경: bg-slate-950 (이미 완료)

중간에 V자로 보이던 다른 색 레이어 제거:

차트를 감싸는 바깥 카드의 배경 색/그라데이션 제거.

필요하면 border 정도만 남기기.

2) 예시
// Before (예시)
<section className="w-full bg-gradient-to-b from-slate-950 to-slate-900">
  <div className="max-w-[1200px] mx-auto bg-slate-900 rounded-3xl ...">
    <Chart ... />
  </div>
</section>

// After
<section className="w-full">
  <div className="max-w-[1200px] mx-auto px-4 lg:px-5 mt-4">
    <div className="rounded-2xl bg-slate-900/80 px-4 py-3">
      <Chart ... />
    </div>
  </div>
</section>


즉, section에는 배경 X,
chart 박스 하나만 살짝 진한 bg-slate-900/80 사용.
