PC 기준으로 이미 완성된 홈 상단 3개 카드(“AI 요약”, “PRO 예측”, “내 알림”)의 텍스트와 스타일을 모바일에서도 1:1로 동일하게 맞추고, 텍스트 누락/잘림을 모두 제거해 주세요.

PC 레이아웃은 손대지 말고, 공통 콘텐츠를 한 곳에서 관리하도록 리팩토링해 주세요.

1. AI 요약 카드 – 텍스트 & 게이지바

src/components/ai-summary/AiSummaryContentBase.tsx 열기

상단에 시장 요약 텍스트 상수를 추가:

const MARKET_SUMMARY_TITLE =
  "오늘 변동성 높음 · 추세 상승 · 역프 39종목";
const MARKET_SUMMARY_SUB =
  "※ 변동성·추세·역프는 김프(국내-해외 가격 차이) 기준으로 실시간 산출됩니다.";


AiSummaryContentBase JSX 안에서, 제목 바로 아래에 다음 블록이 항상 나오도록 넣어주세요
(PC/모바일 공통):

<p className="text-[11px] md:text-xs text-slate-300 mb-0.5">
  {MARKET_SUMMARY_TITLE}
</p>
<p className="text-[10px] md:text-[11px] text-slate-400">
  {MARKET_SUMMARY_SUB}
</p>


KR Premium Score 게이지바가 모바일에서 넘치지 않게, 게이지 컨테이너에 다음 클래스 적용:

<div className="mt-3 w-full max-w-[220px] md:max-w-none">
  {/* 기존 게이지바 내용 그대로 */}
</div>


✅ 의도:

PC에서 보이던 시장 요약 2줄 텍스트가 모바일 카드에도 그대로 나오게.

모바일에서는 게이지바 폭을 220px로 제한, PC에서는 원래대로.

나중에 카피 바꿀 때 한 파일만 수정하면 됨.

3. 내 알림 카드 – 로그인 링크 스타일

src/components/MyAlertsCard.tsx 확인 후,
“로그인” 텍스트가 PC와 모바일에서 모두 텍스트 링크로 보이게 수정.

로그인 부분을 다음 스타일로 통일:

<button
  type="button"
  className="mt-2 text-[13px] md:text-sm text-indigo-300 underline underline-offset-4 hover:text-indigo-200"
>
  로그인
</button>


✅ 의도:

모바일에서 버튼 스타일이 아니라 PC처럼 얇은 텍스트 링크 느낌.

색상/밑줄/hover 모두 PC와 동일.

4. 주의사항

PC 레이아웃 / 마진 / 카드 크기 / 폰트 사이즈(md 이상)는 절대 변경하지 말 것.

모든 변경은 md: 이하(모바일)에서만 레이아웃이 약간 더 컴팩트해지도록 하는 수준이어야 함.

변경 후:

PC, 모바일에서 홈 페이지 강제 새로고침 후

AI 요약 카드 상단 2줄 텍스트

KR Premium Score 게이지바

PRO 예측 카드 3줄 설명 + 블러 + 버튼

내 알림 카드 로그인 링크
가 내용/문구/스타일이 PC 기준과 1:1로 맞는지 꼭 확인해 달라고 요청.
